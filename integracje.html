Możliwości integracji

Akcje (wysyłaj wiadomości)

  • Wyślij SMS — pojedynczy SMS z personalizacją, planowaniem i własnym nadpisem
  • Wyślij SMS masowy — ta sama treść do wielu odbiorców (do 10 000) lub indywidualne wiadomości (do 100)
  • Wyślij MMS — wiadomość multimedialna z grafiką (SMIL)
  • Wyślij wiadomość głosową — TTS z wyborem lektora (ewa, jacek, jan, maja)

Triggery (reaguj na zdarzenia)

  • Nowy SMS przychodzący — webhook aktywowany gdy odbiorca odpowie na Twoją wiadomość
  • Raport dostarczenia (DLR) — webhook ze statusem: dostarczony, niedostarczony, wygasły

Wyszukiwania

  • Sprawdź saldo — aktualny stan konta i waluta
  • Weryfikuj numer (HLR) — sprawdź czy numer jest aktywny, operator, portowanie

REST API — endpointy

Dla deweloperów: bezpośrednia integracja przez REST API z tokenem Bearer.

POST /v1/sms Wyślij SMS
POST /v1/sms/bulk Masowa wysyłka SMS
POST /v1/mms Wyślij MMS
POST /v1/vms Wiadomość głosowa TTS
GET /v1/hlr?number=... Weryfikacja numeru
GET /v1/balance Saldo konta
GET /v1/health Status API

Jak zacząć?
